# hello-py

A simple Python hello world project.

## Requirements

- Python >= 3.14
- [uv](https://github.com/astral-sh/uv)

## Usage

```bash
uv run hello-py
```

## Development

```bash
# Build the package
make build

# Format and auto-fix lint issues
make fix

# Check formatting, lint, and type errors
make check
```
